## F1 VSC参考圈速的确定方法

### 1. 参考圈速的确定依据

根据查询结果，F1 VSC的参考圈速（"正常圈速"）是通过以下方式确定的：

**主要依据：**
- **当前比赛中的最快圈速**：FIA会参考当前比赛（正赛）中车手们已经完成的最快圈速
- **练习赛圈速**：如果比赛圈速数据不足，也会参考练习赛中的最快圈速
- **实时调整**：参考圈速会根据比赛进程实时更新

### 2. 分段计算系统（Marshalling Sectors）

**关键发现：**
- F1赛道被分为3个**调度区（Marshalling Sectors）**
- VSC的参考圈速不是按整圈计算，而是按每个调度区分别计算
- 车手必须在每个调度区内遵守特定的时间限制

**计算方法：**
- 每个调度区都有一个参考时间
- 车手在该调度区内的行驶时间不能低于参考时间
- 参考时间基于该调度区在正常比赛中的最快通过时间

### 3. 具体计算流程

**步骤1：确定基准时间**
- 收集当前比赛中每个调度区的最快通过时间
- 如果数据不足，使用练习赛或历史数据作为参考

**步骤2：应用减速系数**
- 在基准时间基础上增加30-40%的时间作为安全缓冲
- 这意味着车手在每个调度区内的行驶时间要比正常情况下长30-40%

**步骤3：实时更新**
- 系统会根据比赛进程实时更新参考时间
- 如果有车手创造了新的最快圈速，参考时间会相应调整

### 4. 实际应用示例

假设某赛道当前比赛的最快圈速为90秒，分为3个调度区：
- 调度区1：30秒
- 调度区2：25秒
- 调度区3：35秒

在VSC期间，每个调度区的参考时间变为：
- 调度区1：30秒 × 1.3 = 39秒
- 调度区2：25秒 × 1.3 = 32.5秒
- 调度区3：35秒 × 1.3 = 45.5秒

车手必须确保在每个调度区内的行驶时间不低于这些参考时间。

### 5. 技术实现

**监测系统：**
- 使用GPS系统精确测量车手在每个调度区的位置和速度
- 通过赛道上的时间测量点记录车手通过时间
- 实时计算车手是否满足VSC时间要求

**违规处罚：**
- 如果车手在某个调度区内的行驶时间低于参考时间，将受到处罚
- 处罚通常包括时间罚时或通过维修区

## 创建CSV表格

基于以上信息，我们可以创建一个CSV表格，记录VSC参考圈速的确定方法：

```csv
Parameter,Description,Value
Reference_Speed_Source,参考圈速数据来源,Current Race Fastest Lap
Marshalling_Sectors,调度区数量,3
Speed_Reduction_Factor,减速系数,30-40%
Calculation_Method,计算方法,Per Marshalling Sector
Update_Frequency,更新频率,Real-time
Adjustment_Factors,调整因素,Fuel Load, Tire Wear, Track Conditions
Compliance_Monitoring,合规监测,GPS + Timing Loops
Penalty_Threshold,处罚阈值,Exceeding Reference Time
```

### 表格说明：

1. **Reference_Speed_Source**：参考圈速数据来源，通常是当前比赛中的最快圈速
2. **Marshalling_Sectors**：调度区数量，F1赛道通常分为3个调度区
3. **Speed_Reduction_Factor**：减速系数，车手需要减速30-40%
4. **Calculation_Method**：计算方法，按每个调度区分别计算
5. **Update_Frequency**：更新频率，参考圈速会实时更新
6. **Adjustment_Factors**：调整因素，包括燃油负荷、轮胎磨损、赛道状况等
7. **Compliance_Monitoring**：合规监测，使用GPS和时间测量点
8. **Penalty_Threshold**：处罚阈值，超过参考时间将受到处罚

1. 将此CSV保存为`vsc_reference_calculation.csv`
